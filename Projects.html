Academic Projects

1. Fresh Fruits Shelf-Life Prediction using Deep CNN

Developed a deep convolutional neural network (CNN) model to predict the shelf life of fresh fruits and vegetables using temperature simulation data. The system helps in optimizing storage and transportation conditions to reduce food waste.

2. Login and Registration System using Java (JDBC & Servlets)

Designed and implemented a web-based authentication system using Java, JDBC, Servlets, and MySQL. The application supports user registration, login validation, and session handling for secure access control.

3. Student Management System using SQL

Created a relational database system using MySQL to manage student records including personal details, marks, and attendance. Implemented SQL queries such as SELECT, INSERT, UPDATE, DELETE, JOIN, and aggregate functions for efficient data retrieval.
